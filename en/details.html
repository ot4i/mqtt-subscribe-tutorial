Subscribing to receive MQTT events

Learn how to use the MQTT Subscribe node to receive data from an MQTT broker.
OVERVIEW

This tutorial demonstrates a simple message flow that connects to IBM's Internet of Things Foundation and receives events published by a device simulator over the MQTT protocol. The message flow writes the event data to a file.

MQTT is an open protocol for remote communication using a publish and subscribe model. Devices and applications can publish data on named topics and other applications or devices can subscribe to receive data that are published on particular topics.

IBM Internet of Things (IoT) Foundation hosts an MQTT broker on the internet and provides a quickstart demonstration service. This tutorial connects to that service and illustrates how to use a message flow to receive data over MQTT. To publish the data, you will use the device simulator provided by the IoT Foundation.

This tutorial takes approximately 10 minutes.
